Print(警翼手持打印机)

警翼打印机

集成警翼打印机硬件，支持打印 图片+文本+标题

支持平台：

  • Android

[!WARNING]

  • 插件的所有接口在 deviceready 事件后生效;
  • 打印文本内容时换行请使用\r\n
  • 打印图片内容时 最大宽度384(超出将进行自动压缩)，最大高度200（超出讲进行分段打印）

API列表

API 说明
navigator.print.printIma 打印图片
navigator.print.printText 打印普通文本
navigator.print.printTextBig 打印标题
