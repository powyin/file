printTextBig

警翼手持打印机 打印标题

[!TIP|labelVisibility:hidden|iconVisibility:hidden] navigator.print.printTextBit(successCallback, errorCallback, stringContent)

支持平台：

  • Android

参数说明

参数 类型 必填 说明
successCallback Function 是 成功回调函数 return true
errorCallback Function 否 失败回调 打印机初始化问题
stringContent String 否 打印字符串内容

示例代码

// 引用js
<script src='supconit://hcmobile.js'></script>
<script>
    // 监听’deviceready‘事件
    document.addEventListener('deviceready', onDeviceReady, false)
    function onDeviceReady(){

     navigator.print.printTextBig(function (successCallback) {
                 alert(JSON.stringify(successCallback));
             }, function (errorCallback) {
                 alert(JSON.stringify(errorCallback));
             },"请输入标题内容");
    }

</script>

响应示例代码：

true
