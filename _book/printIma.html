printIma

警翼手持打印机 打印图片

navigator.print.printIma(successCallback,errorCallback,base64ImageData)

支持平台：

  • Android

参数说明

参数 类型 必填 说明
successCallback Function 否 回调函数,返回true
errorCallback Function 否 回调函数,返回错误信息，打印机初始化问题
base64ImageData String 是 图片base64字符串

[!WARNING]

  • 插件的所有接口在 deviceready 事件后生效;
  • 最大宽度384(超出将进行自动压缩)，最大高度200（超出讲进行分段打印）

示例代码

// 引用js
<script src='supconit://hcmobile.js'></script>
<script>
// 监听’deviceready‘事件
document.addEventListener('deviceready', onDeviceReady, false)
function onDeviceReady(){
navigator.print.printIma(function (callback) {
  alert(JSON.stringify(callback));
},
function (callback) {
  alert(JSON.stringify(callback));
},
"base64字符串");
}
</script>

响应示例代码：

true
